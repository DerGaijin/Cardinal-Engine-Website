Framework - Core

The Core contains all Types of Containers (Array, Map, Set, Smart Pointers, FunctionRef, String) that are needed for the Basic usage.
It also contains the Logging System used in the Engine.
The Core also has many Math Types (Random, Matrix, Quaternion, Vec2, Vec3, Vec4) that the Engine requires.
The Profiler is also part of the Core.
